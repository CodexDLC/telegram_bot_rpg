# Симбиот и Система Даров

**Статус:** Updated / Code-aligned
**Связанные модули:** `SymbioteService`, `GiftLibrary`, `SymbioteRepo`

---

## 1. Концепция Симбиота (The Symbiote)

**Симбиот** — это био-энергетическая сущность, интегрированная в организм игрока при переходе через Портал. Он выступает проводником магии и интерфейсом ("Системой").

### Единая Прогрессия
Мы отказались от раздельной прокачки "Уровня Симбиота" и "Ранга Дара".
**Сила Симбиота = Ранг Дара.**

* Развивая Симбиота, игрок автоматически усиливает свой Дар и открывает новые способности.
* Симбиот растет, поглощая **Эссенцию Разлома** (Rift Essence) — концентрированную магию, добываемую при закрытии Рифтов или из специальных ресурсов.

---

## 2. Система Даров (The Gift)

Магии в привычном понимании (книги заклинаний) не существует. Есть **Дар** — уникальная мутация, управляемая Симбиотом.

### Правила Дара
1.  **Один на всю жизнь:** Игрок владеет **одним** Даром (Школой). Смена Дара — это полная "пересадка" Симбиота (Hard Reset).
2.  **Источник:** Дар питается Энергией (Energy), которую генерирует Симбиот, и Выносливостью носителя.

### Структура: Школа vs Способности
* **Школа (School):** Глобальный тип энергии (Огонь, Вода, Тьма...).
* **Способности (Abilities):** Конкретные приемы внутри школы, открывающиеся на определенных Рангах.

---

## 3. Ранги Эволюции (Evolution Tiers)

Прогресс Симбиота линеен и разбит на 8 рангов. Переход на новый ранг требует накопления определенного количества `gift_xp`.

| Ранг (Tier) | Титул | Описание и Бонусы | Требование (XP) |
| :--- | :--- | :--- | :--- |
| **1** | **Пробужденный** | Симбиот активен. Доступна базовая пассивная способность. | 0 |
| **2** | **Неофит** | Симбиот адаптируется. Открывается 1-я активная способность. | 10,000 |
| **3** | **Адепт** | Усиление пассивных бонусов. | 50,000 |
| **4** | **Мастер** | Полный контроль. Открывается 2-я активная способность. | 150,000 |
| **5** | **Магистр** | Глобальное усиление параметров школы. | 500,000 |
| **6** | **Архимаг** | Мутация способностей (доп. эффекты в Pipeline). | 1,500,000 |
| **7** | **Абсолют** | Открывается Ультимативная способность (Ultimate). | 3,000,000 |
| **8** | **Вне категории** | Симбиот становится совершенным. Легендарный статус. | 5,000,000 |

## 4. Механика Развития
**Поглощение (Consumption):**
Игрок не получает опыт Дара за использование скиллов.
Для роста нужно "скармливать" Симбиоту ресурсы:
* `Rift Shards` (Осколки Рифта)
* `Essence Cores` (Ядра Эссенции)

Это создает экономическую ценность походов в Рифты.
