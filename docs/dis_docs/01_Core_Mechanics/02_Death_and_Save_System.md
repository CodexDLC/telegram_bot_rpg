# Смерть и Сохранение Прогресса

Используется гибридная система **"Слепок Состояния" (Snapshot)**.

## Концепция "Вылазки" (Run)
1.  **Точка сохранения:** Перед выходом из Безопасной Зоны (в Рифт или Пустошь) создается "Слепок" персонажа (Инвентарь, Опыт, Состояние).
2.  **Риск:** Все, что добыто *во время* вылазки, находится во "временном кармане".
3.  **Смерть:**
    * Если HP падает до 0 внутри Рифта.
    * Игрок "откатывается" к состоянию "Слепка" (в Город).
    * **Штраф:** Потеря всего лута и опыта, полученного *в этой конкретной вылазке*.
    * *Опционально:* Потеря прочности надетого снаряжения.

Это создает азарт (High Risk / High Reward), но не наказывает игрока потерей всего прогресса за месяц (как в полном Permadeath).
