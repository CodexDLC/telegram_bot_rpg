# Chaos Incursion Protocol: The Time Eater (Protocol 666)

**Type:** System Actor / World Event / Garbage Collector
**Trigger:** `job_chaos_check` (10 minutes of inactivity).
**Role:**
1.  **Cleaner:** Техническая зачистка зависших сессий через игровую механику.
2.  **Raid Boss:** Скрытый эндгейм-челлендж для скоординированных групп.

---

## 1. The Entity (Concept & Power Level)
Это сущность класса `Chaos Bringer`.
**Team:** `chaos` (Враждебен ко всем: Red, Blue, Green, Yellow).
**ID:** `-666` (System Reserved).

### Attributes (God Mode)
Мы задаем экстремальные значения базовых атрибутов (Stats), которые через `StatsWaterfallCalculator` конвертируются в "невозможные" боевые параметры.

| Attribute | Value | Effect (via Waterfall) |
| :--- | :--- | :--- |
| **Strength** | **1000** | Колоссальный физический урон. Ваншот любого танка. |
| **Agility** | **1000** | Абсолютная точность и уклонение (99%+). |
| **Endurance** | **1000** | Огромный пул HP (Raid Boss Tier) и резисты. |
| **Intelligence** | **1000** | Смертоносная магия. |
| **Wisdom** | **1000** | Полный иммунитет к магии и дебаффам. |
| **Men** | **1000** | Бесконечная энергия для спама скиллами. |
| **Perception** | **1000** | Видит сквозь невидимость (Stealth Detection). |
| **Charisma** | **1000** | (Lore) Его ужас подавляет волю (Fear Aura). |
| **Luck** | **1000** | Гарантированные криты и проки эффектов. |

### Skills
1.  **Execution:** Базовая атака. Ваншот любой цели, не имеющей механики "Immortal/Cheat Death".
2.  **Entropy Aura (Passive):** Постоянный урон по площади каждый тик (DoT), чтобы бой не затягивался вечно и хилеры не могли держать группу бесконечно.

---

## 2. Incursion Logic (Spawn)

Когда срабатывает триггер (10 минут тишины), **Gateway** выполняет "Hot Join":

1.  **Add to Meta:** Добавляем ID `-666` в список участников.
2.  **Announce:** Системное сообщение в чат боя:
    > *"⏳ Границы реальности истончились. В поисках утраченного времени пришел ОН."*
3.  **Wake Up:** Продлеваем жизнь сессии (`last_activity_ts`) еще на 5 минут (чтобы дать время умереть или победить).

---

## 3. AI Behavior (The Reaper)

Монстр использует агрессивную логику (реализуется через стандартный `AiWorker`):
1.  **Targeting:** Случайная живая цель.
2.  **Speed:** Спам атаками без задержек.
3.  **No Mercy:** Только Атака в уязвимые зоны. Никакой защиты.

---

## 4. The "Exploit" (Risk vs Reward)

Если игроки специально призвали его и смогли убить:
* **Drop:** Уникальный ресурс для крафта (шанс 100%).
* **Achievement:** "Убийца Уборщика".
* **Log:** *"Хаос был изгнан обратно в бездну."*

---

## 5. Fail Safe
Если монстр убил всех:
* Бой завершается. Победитель: `chaos`.
* Сессия удаляется стандартным механизмом завершения боя.
