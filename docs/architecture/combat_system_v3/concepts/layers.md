# Концепция Трех Слоев Боя (Draft)

Боёвка разделена на три ортогональных слоя, которые никогда не смешиваются напрямую.

## 1️⃣ Оружие (Equipment Layer)
**Оружие:**
*   Задаёт ограничения.
*   Вводит штрафы.
*   Определяет возможные зоны поражения.
*   Определяет профиль риска.

> **Принцип:** Оружие не даёт бонусов. Оно всегда делает бой сложнее.

## 2️⃣ Стойка (Stance Layer)
**Стойка:**
*   Описывает, как персонаж использует оружие.
*   Определяет, какие триггеры вообще возможны.
*   Формирует стиль размена.

**Стойка — это:**
*   НЕ класс.
*   НЕ навык.
*   НЕ выбор игрока.

Она определяется автоматически:
*   Конфигурацией рук.
*   Типом экипировки.
*   Контекстом (щит / вторая рука / дальний бой).

## 3️⃣ Навык (Skill Layer)
**Навык:**
*   НЕ открывает действия напрямую.
*   НЕ даёт активных эффектов.
*   НЕ усиливает урон.

**Навык = Контроль вероятностей:**
*   Повышает шанс срабатывания триггеров стойки.
*   Снижает вероятность негативных исходов.
*   Стабилизирует размен.
