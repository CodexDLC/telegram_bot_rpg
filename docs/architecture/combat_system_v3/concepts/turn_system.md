## Ключевые принципы (Summary)

1.  **No Global Turns:** Нет общего "Хода 1", "Хода 2". У каждого игрока свой поток времени.
2.  **Personal Queue:** Враги выстраиваются в очередь к игроку.
3.  **The 60s Rule:** Если игрок не реагирует на атаку 60 секунд -> Автоматический пропуск удара (Force Hit).
4.  **Global Step Counter:** Используется только для синхронизации логов и эффектов (DoT), но не ограничивает действия игроков.

*Подробности см. в Manifesto.*
