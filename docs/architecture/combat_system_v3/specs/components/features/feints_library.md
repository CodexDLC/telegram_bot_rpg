# Feints Library

⬅️ [Назад](../../README.md)

> **Status:** Placeholder / In Development

**Финты (Feints)** — это специальные приемы, зависящие от класса оружия и накопленных тактических токенов.

В отличие от обычных скиллов, финты:
1.  Не имеют кулдауна (ограничены только токенами).
2.  Меняют свойства базовой атаки (например, добавляют `force_crit` или `ignore_block`).
3.  Являются частью системы "Стойка" (Stance Layer).
