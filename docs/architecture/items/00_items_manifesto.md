# Items Manifesto (RBC v3.1)

Этот раздел документации посвящен системе Предметов (Items Layer).

## Структура
*   **Weapon Data:** Структура данных оружия (Trigger, Spread, Accuracy).
*   **Armor Data:** Структура данных брони (Flat Reduction, Penalties).
*   **Consumables:** Расходники.

## Ключевые принципы
1.  **Оружие задает стиль.** Оно определяет доступные финты и триггеры.
2.  **Броня дает вычет.** Она не влияет на шанс попадания (AC), а режет урон (Flat).
3.  **Качество влияет на шанс.** ItemTriggerChance зависит от качества предмета.
