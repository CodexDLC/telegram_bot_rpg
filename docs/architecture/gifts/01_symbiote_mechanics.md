# Symbiote System Specification (RBC v3.1)

## 1. Симбиот как Операционная Система
Игрок не видит "игровой интерфейс". Всё, что он видит — полоски здоровья, мини-карта, описание предметов — это **Дополненная Реальность (AR)**, которую проецирует Симбиот прямо в зрительный нерв носителя.

*   **Лор:** Симбиот — это энерго-информационная форма жизни. Ему нужен носитель (игрок) для существования, а игроку нужен Симбиот для управления Энергией (XP/Пыль).
*   **Кормление:** Поглощение Пыли/Энергии = кормление "пассажира".
*   **Статус:** На старте (Уровень 0-1) — "Спящий режим". Интерфейс глючный, без персонализации.

---

## 2. Уровень 0–1: «Слепая фаза» и Скрытые Токены
Игрок проходит Туториал, думая, что просто бьет мобов. На самом деле, каждое действие анализируется.
Система начисляет скрытые **Affinity Tokens (Токены Склонности)**.

### Как начисляются Токены:
*   **Стиль Боя:**
    *   Танкуешь лицом? -> `+Earth` / `+Light`.
    *   Агрессия? -> `+Fire`.
    *   Кайт/Стелс? -> `+Air` / `+Darkness`.
    *   Ловушки/Умная игра? -> `+Nature` / `+Arcane`.
*   **Типы Разломов:** Закрыл Огненный Рифт -> `+Fire Tokens`.
*   **Диалоги:** Помог? -> `+Light`. Потребовал награду? -> `+Darkness`.

---

## 3. Уровень 2: «Пробуждение» (The Awakening)
Как только накоплено достаточно Энергии, происходит скриптовое событие.

1.  **Первый контакт:** Голос в голове. Симбиот сообщает, что анализ завершен.
2.  **Именование:** Игрок дает имя Симбиоту (используется в системных сообщениях).
3.  **Выдача Дара (The Choice):**
    *   Система выбирает **TOP-3** стихии по накопленным токенам.
    *   Предлагает на выбор только эти три Дара.
    *   **Эффект:** Игрок получает магию, идеально подходящую под его реальный стиль игры.

---

## 4. Этапы Эволюции (Разблокировка слотов)
Симбиот растет вместе с игроком.

### Уровень 2 (Неофит)
*   **Событие:** Пробуждение.
*   **Награда:** Выбор **Первичного Дара** (Основная стихия).
*   **Лор:** Симбиот подключается к моторным функциям.

### Уровень 5 (Магистр)
*   **Событие:** Полная Синхронизация.
*   **Награда:** Открытие **Второй Ячейки** (Sub-Slot).
    *   Можно инсталлировать модуль другой стихии (пассивку).
    *   Или открыть "Ультимативную Способность" текущего дара.
*   **Лор:** Симбиот перестраивает нервную систему. Доступ к "Глубокому Лору".

### Уровень 7 (Абсолют)
*   **Событие:** Слияние.
*   **Награда:**
    *   **Трансформация (Avatar Mode):** Временное превращение в чистую энергию.
    *   **Смена Дара (Трансплантация):** Возможность сменить "полярность" (основную стихию).
*   **Лор:** Граница между носителем и Симбиотом стирается.
