# Skills Manifesto (RBC v3.1)

Этот раздел документации посвящен системе Навыков (Skills Layer).

## Структура
*   **Reference:** Референсы и идеи из других игр.
*   **Mechanics:** Как навыки влияют на бой (контроль вероятностей).
*   **Progression:** Как навыки прокачиваются.
*   **Data Structure:** JSON-структура навыков.

## Ключевые принципы
1.  **Навык = Контроль вероятностей.**
2.  Навык не дает активных кнопок (обычно).
3.  Навык стабилизирует размен.
