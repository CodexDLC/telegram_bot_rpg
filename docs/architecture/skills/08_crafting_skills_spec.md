# Crafting Skills Specification (RBC v3.1)

## CRAFTING (Крафт)
Полный производственный цикл.

*   **weapon_craft:** Оружейник (Крафт пушек).
*   **armor_craft:** Бронник (Крафт защиты).
*   **jewelry_craft:** Ювелир (Бижутерия/Гемы).
*   **alchemy:** Алхимия (Расходники/Баффы).
*   **science:** Инженерия (Гаджеты/Ловушки/Схемы).
*   **artifact_craft:** Зачарование (Магический апгрейд).
