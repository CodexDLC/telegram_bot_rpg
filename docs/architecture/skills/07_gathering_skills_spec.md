# Gathering Skills Specification (RBC v3.1)

## GATHERING (Добыча)
Основа экономики.

*   **mining:** Горное дело (Руда/Камень).
*   **woodcutting:** Лесоруб (Дерево/Уголь).
*   **skinning:** Снятие шкур (Кожа/Мех).
*   **herbalism:** Травничество (Ингредиенты зелий).
*   **hunting:** Охота (Следопыт/Мясо).
*   **archaeology:** Археология (Артефакты/Древности).
